Welcome to vrsyslog

vrsyslog is a Graphical User Interface for rsyslog. It performs basic analisys based on the log data, and let's you access raw log data.

Installation

Place the public_html folder in a web accessible directory (make sure the web server process has write access to the public_html/assets and public_html/protected/runtime folders).

Edit the configuration file public_html/protected/vrsyslog.ini to suite your needs.

Read INSTALLATION_instructions.txt for more details.
